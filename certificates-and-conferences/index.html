The Certified Kubernetes Application Developer (CKAD) exam certifies that users can design, build, configure, and expose cloud native applications for Kubernetes.

Certified Kubernetes Administrator

The Certified Kubernetes Administrator (CKA) program provides assurance that CKAs have the skills, knowledge, and competency to perform the responsibilities of Kubernetes administrators.

KubeCon | CloudNativeCon - Virtual - Europe 2020

In August 2020, I participated at the first virtual KubeCon | CloudNativeCon. My main focus was on security and software architecture.

I heard interesting talks about security screening on container and cluster level, cluster conformance testing and how to force security policies via kubernets built-in features and webhooks.

Regarding system architecture, I learned a lot about how to use operators and CRDs to automate setup of commonly used components and infrastructure.
