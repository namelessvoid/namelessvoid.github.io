qrwar

qrwar (quad-ruled war) is an isometric turn based tactic game which takes place on a board subsectioned into squares. It is a little spare time project writen in C++ using SFML to deepen my knowledge on 2D graphics and complex software architectures for game engines.

  • Source Code

Gameplay

In qrwar two commanders of opposing armies have battles on a huge and customizable battle field. The gameplay is based on games like Advanced Wars by Nintendo but with a very simplyfied approach: It should be possible to play this game on a piece of squared paper without complex rulesets and combat systems.

Features:

  • Turn-based tactics.
  • 2-Player hot seat mode: Challange a friend (or foe) at one PC.
  • Customize the landscape of the battle field.
  • Simpliest game mechanics.
  • And even more content to come!

Before the battle, both players get their troops deployed on the highlighted deployment areas.

The attacking yellow soldiers are charging the castle and overcome the walls with ladders.

A featured map editor allows to create custom maps.
