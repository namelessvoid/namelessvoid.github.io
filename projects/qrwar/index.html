qrwar

qrwar (quad-ruled war) is a pen & paper like turn based tactic games which takes place on a board subsectioned into squares. It is a little project to learn 2D programming especially with SFML 2.x.

qrwar is written in C++ using SFML for graphics output. SFGUI is used for the main menus whereas the ingame menus are created by a custom framework called nameless-gui I developed for this purpose to satisfy my needs.

Note: This project is still under development so currently no binary packages are provided.

  • Source code >

Gameplay

In qrwar two commanders of opposing armies have battles on a huge and customizable battle field. The gameplay is based on games like Advanced Wars by Nintendo but with a very simplyfied approach: It should be possible to play this game on a piece of squared paper without complex rulesets and combat systems.

Features

  • Turn based tactics.
  • 2-Player hot seat mode: Challange a friend (or foe) at one PC.
  • Customize the landscape of the battle field.
  • Simpliest game mechanics.
  • And even more content to come!
