Generic n-body simulation engine with GPU support (2014-2015)

My bachelor I have written at the University of Applied Sciences Augsburg.

Particle and n-body systems have many use cases in scientific researche and computer graphics. This thesis proposes a generic engine to model, update and render these systems. OpenGL compute shaders are used to perform computations on the highly parallel architecture of graphics cards to speed up the simulation. Read more...

C++ OpenGL 4.3

qrwar (2013 - 2014)

qrwar (quad-ruled war) is a pen & paper like turn based tactic games which takes place on a board subsectioned into squares. It is a little project to learn 2D programming especially with SFML 2.x. Read more...

Blomma Graphs is a tool suite for the generation, scheduling and evaluation of task graphs. It was created by me and four more students of University of Applied Sciences, Augsburg, as an external project at Linnaeus University, Växjö, Sweden. Read more...
